# 研究备忘录

## 背景
NVDA在2025年5月28日公布FY26Q1业绩，收入441亿美元超预期，同比增长69%。Blackwell架构加速放量，四大云厂商2025年资本开支合计超3000亿美元，需求端未现松动。但H20出口限制导致45亿美元减值，Q2指引中已计入约80亿美元H20收入损失，市场需消化这一冲击。

## 看多逻辑
Q2收入指引450亿美元，在扣除约80亿美元H20损失后仍超Street预期的430亿，说明Blackwell需求足以填补中国业务缺口。剔除H20减值后非GAAP毛利率71.3%，Q2指引非GAAP毛利率72.0%，管理层承诺本财年末回归mid-70s区间。四大超大规模云厂商2025年资本开支合计超3000亿美元且仍在加码，Meta上调指引至640-720亿美元，验证AI基础设施需求未减。

## 看空逻辑
AI资本开支消化风险正在累积：TD Cowen发现微软放弃约两GW数据中心项目，AWS也被报道暂停部分租赁谈判。DeepSeek证明算法进步可大幅降低训练算力需求，削弱AI需求到GPU销量的映射。GPU集群在5-6年折旧周期下对仍薄的AI应用收入构成盈利门槛，若AI货币化不及预期，2026年可能进入类似2019年的资本开支消化期。客户集中度高且中国业务基本被切断，增加下行风险。

## 结论
NVDA FY26Q1业绩和Q2指引证明Blackwell需求足以覆盖H20损失，短期基本面强劲且超大规模客户资本开支支撑2025年增长。但估值已反映高增长预期，AI资本开支消化风险和算法效率提升构成中期不确定性。预计40个交易日内股价温和上行，但上行空间受估值和宏观风险限制。
