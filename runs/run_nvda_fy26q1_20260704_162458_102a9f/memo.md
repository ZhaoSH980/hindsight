# 研究备忘录

## 背景
NVDA即将于2025年5月28日发布FY2026第一季度财报，股价已从4月初低点大幅反弹至132.64美元。市场关注焦点集中在H20出口限制带来的55亿美元减值、Blackwell架构产能爬坡以及超大规模厂商资本支出是否持续。四大云厂商2025年合计资本支出暗示将超过3000亿美元，但中国出口管制和毛利率压力构成主要下行风险。

## 看多逻辑
最强看多逻辑在于超大规模厂商的资本支出并未动摇——Meta甚至上调了2025年资本支出指引至640-720亿美元，微软、谷歌和亚马逊合计暗示超过3000亿美元的年度投入，且微软明确表示AI算力仍处于供不应求状态。Blackwell架构在FY2025第四季度已实现110亿美元收入，为公司历史上最快产品爬坡，TSMC的CoWoS先进封装产能持续扩张以缓解瓶颈。若Q2指引在失去中国H20业务的情况下仍能达到或超过450亿美元，将有力证明Blackwell需求足以填补中国市场缺口。

## 看空逻辑
最强看空逻辑在于AI资本支出周期可能接近顶部：TD Cowen的供应链调研发现微软放弃了约2GW的数据中心项目，富国银行报告显示AWS也暂停了部分租赁谈判。DeepSeek证明算法进步可以大幅降低单位算力需求，削弱了AI需求到GPU出货量的映射关系。H20出口限制已实质上移除了英伟达大部分中国数据中心业务，约170亿美元的年收入面临风险，且10-K显示客户集中度较高，个别客户反复超过收入的10%。

## 结论
短期内NVDA基本面依然强劲，超大规模厂商资本支出确认和Blackwell爬坡为5月28日财报提供了坚实支撑。但H20减值、毛利率过渡压力以及2026年潜在的资本支出消化风险使上行空间受限。预计财报前后股价将维持高波动，适度看多但不确定性较大。
